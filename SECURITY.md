# Security Policy

## Supported Release Path

Only `main` is supported for production. Production deploys must come from the checked-in GitHub Actions workflow and the VPS release directories under `/srv/robotronics/frontend`.

## Reporting A Vulnerability

Do not open a public issue for security reports.

Use GitHub private vulnerability reporting when it is available for the repository. If that is not available, contact the repository administrators directly through a private channel and include:

- a short impact summary
- exact steps to reproduce
- affected environment or branch
- any logs, screenshots, or request samples needed to verify the issue
